Change the setup of a field

The "setup" of a field includes field visibility, whether the field can have multiple lines, the width of the field and the input systems available for data.

In the Dictionary Configuration view, do these steps in the Fields tab:

  1. In the left pane, click the field you want to change.

    • If the field you want is not displayed, click (Show All Fields).

  2. In the right pane, do these steps:

    • Select () Hidden if empty if you want to hide the field when it is empty.

    • Select () Display as multi-line text if you want ____________.

    • Edit the number in the Width box to change the width of the field.

    • Select () each input system you want to display in the field; clear () the check box for other input systems that you do now want to appear in the field.

    • Click an input system, and then click the (up arrow) or (down arrow) to change the order of the input systems.

    • Click (Apply).

Note

  • You can click the Edit Input Systems link to open a pane with additional controls. See: Edit Input Systems.
