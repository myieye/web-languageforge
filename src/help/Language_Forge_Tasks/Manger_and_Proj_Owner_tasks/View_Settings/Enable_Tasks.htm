Enable Tasks

Do these steps in the View Settings view. This view shows a tab for each role.

  1. Click the tab for the role you will change.

  2. If the pane that shows tasks is not shown, click the Enable Tasks header.

  3. In the left pane, do these steps:

    • Select () each task you want to be enabled for this role.

    • Clear () each task you do not want enabled.

  1. Repeat any of the above steps for other roles, as desired.
