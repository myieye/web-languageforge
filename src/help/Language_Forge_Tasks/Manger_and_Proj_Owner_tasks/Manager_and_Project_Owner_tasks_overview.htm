Manager and Project Owner tasks overview

A person who has the Manager and Project Owner role can start a new project and do any of the tasks people in other roles can do.

In addition, as the person with the most enabled tasks, he or she can do any of the tasks in any of these areas:

  • Dictionary Configuration

  • View Settings

  • Import/Export

  • User Management

  • Project Settings

  • Manage Project
