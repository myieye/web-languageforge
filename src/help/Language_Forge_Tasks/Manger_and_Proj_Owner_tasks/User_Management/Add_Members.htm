Add Members

Do these steps in the User Management view.

  1. Click the Members tab.

  2. Click (Add Members).

A text box and a button appear.

  1. Click the text box, and then type the name of the person you want to add.

A drop-down list of names appears if the name you typed matches names in the list.

    • If the person you want to add appears in the drop-down list, click that person's name. Click (Add Existing User).

    • If the person you want to add does not appear in the drop-down list, click (Create New User).

A row is added to the table of members.

  1. Choose a role for the member.
