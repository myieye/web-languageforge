Start or Join a New Project

  1. In the My Projects view, click (Start of Join a New Project).

  2. Click the button for the kind of project you want to start or join.

The Start or join a Web Dictionary Project view appears. It has these three steps: Name, Initial Data and Verify.

  1. In the Name step, click the Project Name box and then type a name of the project. Click (Next).

  2. In the Initial Data step, use drag-and-drop or click (Browse) to import a compressed (zipped) folder that contains the whole project you need to import. Then click (Next).

    • If you do not have any initial data, click (Skip).

  3. In the Verify/Language step, click (Select), and choose primary input system language.

    • Click (Add) add a new input system.

  4. Click (Dictionary) to open the new dictionary project in Language Forge.

Tip

  • Currently, you can only click Dictionary: .

  • Click (Back) to move back to a previous step.
