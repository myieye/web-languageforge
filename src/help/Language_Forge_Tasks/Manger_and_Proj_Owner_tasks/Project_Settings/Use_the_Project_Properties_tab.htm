Use the Project Properties tab

Do any of these steps in the Project Settings view.

  • Edit the project name:

    • Click and edit the project name in the Project Name box.

  • Change the default User Interface language:

    • Click the Default Interface Language box and then click the language you want

  • Display the name of the project owner on the website:

    • Select () Featured On Website.

  • Click to save your changes.
