Language Forge Tasks overview

The tasks that can be done in Language Forge depend on the role of the person and which tasks are enabled.

In the set of overview topics below, tasks that can be done are grouped under by role. However, because the various roles may have different tasks enabled, they tasks are grouped under the lowest role in the hierarchy (has the fewest possible set of enabled tasks) in which the task is likely to be done.
