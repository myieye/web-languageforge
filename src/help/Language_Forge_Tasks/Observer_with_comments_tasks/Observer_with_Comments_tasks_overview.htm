Observer with Comments tasks overview

The tasks that an observer with comments do are controlled by the person who has one of these roles: Manager and Project Owner or Manager.

Typically, observer with comments do these tasks:

  • Any of the observer tasks.

  • Add a comment to a lexical entry.

  • Edit an existing comment.

  • Reply to an existing comment.

  • Mark a comment as resolved () or as a task to do ().
